```bash
| - board

|	| - ...
| - kernel

| - arch
| 	| - ...

|	| - ...

```

- board
  - 放置说明 board 测试相关的内容
- pic
  - 子模块目录放置的地方
- arch
  - 放置 arch 测试相关的内容
- litelibc 
  - 放置 litelibc 测试相关的内容

- kernel
  - 放置 kernel 测试相关的内容

- rpmsglite 
  - 放置 rpmsglite 测试相关的内容


## 模块说明

- oerv-rtos 入门测试模块用于检测是否建立起正常的执行流，以及是否能理解当前执行流每一步的作用
- oerv-rtos 入门测试模块划分了针对不同模块的测试
